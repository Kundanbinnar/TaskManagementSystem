## Task Management System
A web-based Task Management System built with HTML, CSS, JavaScript (jQuery), and Chart.js, designed to help users manage and track tasks efficiently. This project allows users to view, add, edit, and delete tasks, while also providing a visual summary of task statuses through interactive charts.

## Features
- Add, edit, delete tasks
- Task status tracking: Pending / In Progress / Completed
- Dashboard with Pie & Bar charts for task distribution
- User-based task management
- Task list with search, sort & pagination

## Technologies Used
- Backend: Java, Hibernate, Springboot
- Frontend: HTML, CSS, Bootstrap 5, JavaScript, jQuery
- Charts & Visualization: Chart.js
- Data Handling: DataTables.js
- Backend: REST API / JSON data for tasks
